Modul 08: Importe, Module und Projekte

Ziel des Moduls

Die Teilnehmenden lernen, wie .ns-Programme ueber mehrere Dateien organisiert werden und warum Modularisierung auch in Nova Language fuer Lesbarkeit und Wartbarkeit entscheidend ist.

Lernziele

  • import lesen und einsetzen
  • gemeinsame Bausteine auslagern
  • Dateien sinnvoll benennen und strukturieren
  • ein kleines Mehrdatei-Projekt ueberblicken

Beispiele

  • 08_import_shared.ns
  • 08_import_main.ns

Fachlicher Kern

Sprachunterricht auf Grundkursniveau sollte spaetestens an dieser Stelle zeigen, dass gutes Programmieren immer auch gute Struktur bedeutet. Nova Language hat dafuer den gleichen Anspruch wie andere Sprachen:

  • nicht alles in eine Datei
  • Bausteine gezielt auslagern
  • Wiederverwendung und Lesbarkeit verbessern

Live-Demo

@'
from pathlib import Path
from nova.runtime.runtime import NovaRuntime
path = Path(r".\NovaLanguage\Beispiele\08_import_main.ns").resolve()
source = path.read_text(encoding="utf-8")
with NovaRuntime() as runtime:
    runtime.load(source, source_name=str(path), base_path=path.parent)
    print("Mehrdatei-Programm erfolgreich geladen.")
'@ | python -

90-Minuten-Ablauf

Phase 1: Problemstellung

  • Wann wird eine Datei zu gross
  • Warum Wiederverwendung wichtig ist

Phase 2: Importsyntax

  • importierte Datei
  • gemeinsame Deklarationen
  • Hauptdatei mit Flow

Phase 3: Projektstruktur

  • Shared-Datei
  • Main-Datei
  • Benennungskonventionen

Phase 4: Praxis

  • Agent und Dataset in Shared-Datei
  • Flow in Main-Datei

Zentrale Begriffe

  • Import
  • Modul
  • Projektstruktur
  • Wiederverwendung
  • Hauptdatei

Typische Fehler

  • zu grosse Sammeldateien
  • unklare Dateinamen
  • Zirkularitaet oder unnoetige Abhaengigkeiten

Uebungen

  1. Lagere einen weiteren Baustein in die Shared-Datei aus.
  2. Erstelle eine zweite Main-Datei mit anderer Nutzung derselben Bausteine.
  3. Benenne die Dateien fuer ein Schulungsprojekt um.

Hausaufgabe

Baue ein Mini-Projekt mit mindestens zwei .ns-Dateien, einer Shared-Datei und einer Hauptdatei.
