Modul 04: Listen, Maps und Dictionaries

Ziel des Moduls

Die Teilnehmenden lernen, wie Nova Language strukturierte Sammlungen verarbeitet und wie Listen- und Dictionary-Denken mit dataset und py.exec zusammenhaengt.

Lernziele

  • Listen von Objekten verstehen
  • Maps beziehungsweise Dictionary-Strukturen bilden
  • Werte aggregieren und filtern
  • Datenmodelle fuer echte Aufgaben entwerfen

Beispiel

- 04_listen_maps_und_dictionaries.ns

Fachlicher Kern

Wer Nova Language lehren will, muss den Uebergang von linearen Einzelwerten zu strukturierten Sammlungen sicher gestalten. Dieses Modul ist das Pendant zu Python-Einheiten ueber Listen, Dictionaries und Schleifen.

Nova Language loest dieses Thema im Grundkurs ueber:

  • Datensammlungen in dataset
  • Transformationen in py.exec
  • interpretierbare Zwischenergebnisse ueber Aliase

Live-Demo

python nova_shell.py -c "ns.graph .\NovaLanguage\Beispiele\04_listen_maps_und_dictionaries.ns"
ns.run .\NovaLanguage\Beispiele\04_listen_maps_und_dictionaries.ns

90-Minuten-Ablauf

Phase 1: Wiederholung Datenstrukturen

  • Liste von Eintraegen
  • jeder Eintrag als Map

Phase 2: Dictionary-Denken

  • Zuordnung name -> score
  • Warum Maps fuer Lookup und Berichte hilfreich sind

Phase 3: Ableitungen bilden

  • aktive Teilnehmer zaehlen
  • Werte in neue Strukturen ueberfuehren

Phase 4: Transfer in Domänenbeispiele

  • Produkte
  • Sensorwerte
  • Kursteilnehmer

Zentrale Begriffe

  • Liste
  • Map
  • Dictionary
  • Key
  • Value
  • Aggregation

Typische Fehler

  • unvollstaendige oder inkonsistente Datensaetze
  • vergessen, welche Struktur aus einem Schritt herauskommt
  • Verwechslung von Liste und Map in der weiteren Verarbeitung

Uebungen

  1. Ergaenze eine Berechnung fuer den hoechsten Score.
  2. Gib nur aktive Teilnehmer in einer separaten Liste aus.
  3. Erweitere die Datensaetze um ein Feld group.

Hausaufgabe

Modelliere ein dataset fuer ein Lager, eine Klasse oder einen Projektstatus und erzeuge daraus mindestens eine Map und eine Zusammenfassung.

Transferfrage

Warum ist das Verstaendnis von Listen und Dictionaries die Voraussetzung dafuer, spaeter Agenten- und Service-Programme sinnvoll zu lesen
