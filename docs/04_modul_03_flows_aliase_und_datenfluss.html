Modul 03: Flows, Aliase und Datenfluss

Ziel des Moduls

Die Teilnehmenden verstehen, wie Daten in einem flow von Schritt zu Schritt weitergereicht werden und welche Rolle Aliase fuer Lesbarkeit und Denkstruktur spielen.

Lernziele

  • einen mehrstufigen flow lesen
  • den Pfeil -> als Aliasbildung verstehen
  • Zwischenergebnisse sinnvoll benennen
  • Datenfluss statt nur Befehlsabfolge erkennen

Beispiel

- 03_alias_und_datenfluss.ns

Fachlicher Kern

Dies ist das erste Modul, in dem sich Nova Language klar von einem klassischen Skriptkurs unterscheidet. Lernende sollen verstehen:

  • Ein Flow ist eine Folge semantischer Verarbeitungsschritte.
  • Ein Alias wie -> total benennt das Ergebnis eines Schritts.
  • Gute Aliasnamen sind fuer Lesbarkeit und Wartung entscheidend.

Didaktischer Vergleich mit Python

Python:

total = sum(item["value"] for item in numbers)
summary = f"Gesamtsumme: {total}"

Nova Language:

py.exec "sum(item['value'] for item in _)" numbers -> total
py.exec '"Gesamtsumme: {}".format(_)' total -> summary

Live-Demo

python nova_shell.py -c "ns.graph .\NovaLanguage\Beispiele\03_alias_und_datenfluss.ns"
ns.run .\NovaLanguage\Beispiele\03_alias_und_datenfluss.ns

90-Minuten-Ablauf

Phase 1: Von Daten zu Verhalten

  • dataset als Eingang
  • flow als Verarbeitung

Phase 2: Aliasbildung erklaeren

  • Was bedeutet -> total
  • Warum total besser ist als x1

Phase 3: Datenfluss visualisieren

  • Eingang numbers
  • Transformation zu total
  • Transformation zu summary
  • Ausgabe

Phase 4: Eigene Varianten

  • Durchschnitt statt Summe
  • Maximum statt Summe
  • Textuelle Ergebnisdarstellung anpassen

Zentrale Begriffe

  • Alias
  • Datenfluss
  • Transformation
  • Zwischenergebnis
  • Lesbarkeit

Typische Fehler

  • Aliasnamen ohne Aussagekraft
  • Eingabe _ und benannte Aliase verwechseln
  • zu viel Python-Logik in einen einzigen Schritt packen

Uebungen

  1. Baue aus dem Beispiel eine Durchschnittsberechnung.
  2. Fuege einen weiteren Schritt hinzu, der die Anzahl der Werte berechnet.
  3. Benenne die Aliase so um, dass ein Dritter den Flow sofort versteht.

Hausaufgabe

Schreibe einen flow, der aus einem dataset mit Punkten die Gesamtsumme und einen kurzen Bericht erzeugt.
